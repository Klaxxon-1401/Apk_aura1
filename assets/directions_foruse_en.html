Aura IR Smart Remote Control Instructions

 

 

1 .Insert the Remote-control Accessories

Android system( MicroUSB and Type-C version)Open the APP after download, insert the remote-control accessories into the mobile phone's MicroUSB interface or Type-C interface.(if it is unrecognized,please check and confirm if the mobile phone support OTG

 

2 .Adapt Infrared Appliances

After confirm that the remote-control accessories are connected,open the APP and enter the remote-control interface,click the “＝”on the top left corner, click “＋”，select the type of the appliance in the "add control" interface,and then select the brand of the appliance.

Automatic matching: Please keep the surrounding environment quiet when matching, and aim the remote control head at the air conditioner and within 4 meters of the air conditioner.

Manual matching: When matching, please aim the remote control head at the home appliance and press the button on the interface. When the home appliance responds, it will respond. If the home appliance responds with the button on the matching interface, the complete remote control interface will come out. If you click the button and the home appliance does not respond, then click the "No Response" button to skip to the next code library and perform the same match.

In addition,you can also input the appliance model and search it directly in the search box on the ＂add control" interface,by which you can find the remote-control code quickly.

 

3 .Save the remote-control unit

After adapting the appliance as per instruction,save the remote-control unit and it can be used directly in the next time.

 

 

 

Frequently Asked Questions

 

1.Can this product remotely control all the appliances?

This product supports all the remote control and learning of infrared appliances,but it does not support non-infrared controlled appliances(such as Bluetooth remote control or appliances of 2.4 GHz). Its own code data base has a level of over 90% compatibility with appliances.

 

2.Does this product need to aim directly at the remote control of appliance?

Yes.The remote control theory of this product is the same as that of appliance' s own remote control. It needs to aim directly at the appliance and cannot have obstructions in the middle.

 

3.Is this product available throughout the world?

This product is supported by global code data base, and the APP language is synchronized with the system of mobile system. It supports Simplified Chinese, Chinese Traditional,English ,Korean ,

Japanese,French ,German,Polish , Portuguese and Hindi, etc. (Part of functions are only for Chinese Mainland)

 

4.Why is there no response after inserting my Andriod phone or not controlling remotely?

If there is no response after inserting this product, first please make sure whether your phone supports the OTG function,and you may make inquiries to the mobile phone producer. If it supports the OTG function, please make sure the OTG function has been started in mobile phone settings. (Some mobile phones' OTG functions may be turned off acquiescently during leaving the factory,which need to be turned on manually). If the mobile phone can recognize the appliance but cannot control remotely, please choose the device as “USB model” . Besides, please make sure the mobile phone itself has no infrared function built in. Some smartphones integrate the infrared function into themselves,so these kinds of mobile phones do not need this product ( such as, Huawei Glory 6, Millet 4, etc.).
